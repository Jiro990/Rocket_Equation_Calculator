# 🚀 Rocket Equation Calculator

## Overview
A Python calculator that solves the Tsiolkovsky Rocket Equation to calculate rocket performance values.

## Features
- Calculate Delta-v (Δv)
- Calculate Initial Mass (m0)
- Calculate Final Mass (mf)
- Calculate Specific Impulse (ISP)
- Display information about rocket equation variables

## Formula

Δv = Isp × g0 × ln(m0 / mf)

Where:

- Δv = Change in velocity (m/s)
- Isp = Specific impulse (seconds)
- g0 = Standard gravity (9.80665 m/s²)
- m0 = Initial mass (kg)
- mf = Final mass (kg)

- Python 3.x
